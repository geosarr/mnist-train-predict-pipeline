WELCOME !!!

This is a FastAPI application that exposes two endpoints to train and predict the classical MNIST dataset with a MultiLayer Perceptron.



It supports token generation after authentification, each token expires after 5 minutes. They have to be renewed to access the ressources. An authentification is done using a username and a password.



There are only two users John Doe (with username: johndoe and password: very_secret) and Alice (with username: alice, I let you imagine the password of Alice). John Doe can only run predictions but cannot train a model, only Alice can do both. There are two scopes to handle the permissions: "training:run" and "prediction:run", Alice has both but John has only the latter.



You can interact with the API by going to the /docs path of present URL.
